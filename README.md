# Add Time Intervals

時間の区間の総和を計算できます。

## 技術スタック

| 技術            | 用途                     |
| --------------- | ------------------------ |
| pnpm            | パッケージマネージャー   |
| Vite            | ビルド                   |
| React           | UIライブラリ             |
| Tailwind CSS    | スタイリング             |
| shadcn/ui       | コンポーネントライブラリ |
| Prettier        | フォーマッター           |
| ESLint          | リンター                 |
| Husky           | Git hooks                |
| lint-staged     | フォーマット・リント     |
| Vitest          | テストランナー           |
| Testing Library | インタラクションテスト   |
